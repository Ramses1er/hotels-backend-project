# BACKEND HOTEL PROJECT

# 1. CONTEXTE

Ce projet est le projet backend pour la gestion des hotels

# 2. Les acteurs du systeme

- Utilisateurs
- Administrateurs

# 3. Use case

En tant qu'un utilisateur je dois :

- Creer un compte
- Me connecter
- Me deconnecter
- Voir la liste des hotels et de toutes les informations les concernants
- Je peux ajouter un hotel dans favoris
- Je peux supprimer un hotle dans favoris

En tant que administrateur :

- Je peux creer mon compte
- Me connecter
- Me deconnecter
- Consulter la liste des utilisateurs
- Modifier un utilisateur
- Suspendre un utilisateur
- bloquer et debloquer un utilisateur
- Ajouter un hotel
- Supprimer un hotel
- Ajouter un arrondissement
- Modifier un arrondissement
- Ajouter un quartier
- Modifier un quartier

# 4. Elements complementaires

- Technologies utilisee : JavaScript : NodeJs (Express)

- Base de donnees : postgreSQL

- ORM : drizzle

- Gestionnaire de package : bun

- Validation des donnees : zod
